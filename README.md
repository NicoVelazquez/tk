# tk
## Install

    $ git clone https://github.com/NicoVelazquez/tk.git
    $ cd tk
    $ npm install

## Configurations

Create file `.env` and include the following variables.

```
    MONGO_URI
```

Edit development, production and testing configuration files in `./config` 

```
    port
```

## Run in dev mode

    $ npm run dev

## Run tests

    $ npm run test
